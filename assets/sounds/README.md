Place your local audio files in this folder with these filenames:

- `assets/sounds/place.mp3`
- `assets/sounds/flip.mp3`
- `assets/sounds/win.mp3`
- `assets/sounds/button.mp3`
- `assets/sounds/music.mp3`

The game code is already prepared to use them through `lib/audio.ts`.
Once the files are added, replace the `null` placeholders in that file with the matching `require(...)` lines.
